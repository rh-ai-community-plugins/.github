# Community Plugins for Red Hat AI

> AI moves fast. Enterprise platforms need stability. Community Plugins bridge the gap.

This organization hosts the repositories behind the Red Hat AI Community Plugins.
Plugins add new, possibly ephemeral capabilities without touching the core supported platform. Simple install, simple removal.

- **[Charter](https://github.com/rh-ai-community-plugins/charter)** — governance, plugin spec, submission process, and the plugin catalog
- **[hello-plugin-world](https://github.com/rh-ai-community-plugins/hello-plugin-world)** — reference implementation to get started

> **Work in progress** — we're actively building this out. Expect changes.
